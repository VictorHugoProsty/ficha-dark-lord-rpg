Como deixar essa ficha à mão no celular
Tudo que você digita aqui já salva sozinho neste navegador, neste aparelho — não precisa clicar em nada para isso.

Para levar os dados preenchidos para o celular (ou mandar para outra pessoa), use o botão Link com dados: ele copia um link que já abre com a ficha preenchida, sem precisar de arquivo.

O botão Exportar baixa um arquivo .json como backup — use Importar para carregar esse arquivo depois.

No iPhone (Safari): toque em Compartilhar → Adicionar à Tela de Início.
No Android (Chrome): toque no menu ⋮ → Adicionar à tela inicial / Instalar app.
